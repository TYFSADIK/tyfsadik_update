Step-by-Step Process

01
Read and Write Text Files

Use context managers for safe file handling.

# Read entire file
with open('data.txt', 'r', encoding='utf-8') as f:
    content = f.read()

# Read line by line (memory-efficient for large files)
with open('data.txt') as f:
    for line in f:
        print(line.strip())

# Write
with open('output.txt', 'w') as f:
    f.write('Hello, file!\n')

# Append
with open('log.txt', 'a') as f:
    f.write(f'{datetime.now()}: event occurred\n')
02
JSON and CSV

Work with structured data formats.

import json, csv

# JSON write/read
data = {'name': 'Alice', 'scores': [95, 87, 92]}
with open('data.json', 'w') as f:
    json.dump(data, f, indent=2)
with open('data.json') as f:
    loaded = json.load(f)

# CSV
with open('records.csv') as f:
    reader = csv.DictReader(f)
    for row in reader:
        print(row['name'], row['score'])

Challenges & Solutions

  • Not using 'with' can leave files open on exception — always use with
  • json.load() vs json.loads() — load() from file, loads() from string

Key Takeaways

  • Specify encoding='utf-8' explicitly — default encoding varies by OS
  • pathlib.Path is more readable than os.path for file operations
