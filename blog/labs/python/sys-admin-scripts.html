System Administration Scripts

Intermediate Python

Objective

Write Python scripts to automate system administration tasks: disk monitoring, process management, and service checks.

Tools & Technologies

Step-by-Step Process

01
subprocess — Run Commands

Execute system commands and capture output.

import subprocess

# Run command, capture output
result = subprocess.run(
    ['systemctl', 'status', 'nginx'],
    capture_output=True, text=True
)
print(result.stdout)
print(result.returncode)  # 0 = success

# Raise exception on non-zero exit
subprocess.run(['apt', 'update'], check=True)

# Shell command (avoid with user input — injection risk)
out = subprocess.getoutput('df -h')
02
psutil — System Monitoring

Monitor CPU, memory, disk, and processes.

import psutil

# CPU
print(f'CPU: {psutil.cpu_percent(interval=1)}%')

# Memory
mem = psutil.virtual_memory()
print(f'RAM: {mem.percent}% used ({mem.used//1024//1024} MB)')

# Disk
disk = psutil.disk_usage('/')
print(f'Disk: {disk.percent}% used')

# Alert if disk > 80%
if disk.percent > 80:
    print('ALERT: disk usage critical')
03
Health Check Script

Combine subprocess and psutil for a system health report.

#!/usr/bin/env python3
import psutil, subprocess, json
from datetime import datetime

def check_service(name):
    r = subprocess.run(['systemctl','is-active',name], capture_output=True, text=True)
    return r.stdout.strip() == 'active'

report = {
    'timestamp': datetime.now().isoformat(),
    'cpu_pct': psutil.cpu_percent(1),
    'mem_pct': psutil.virtual_memory().percent,
    'disk_pct': psutil.disk_usage('/').percent,
    'services': {
        'nginx': check_service('nginx'),
        'ssh': check_service('ssh'),
    }
}
print(json.dumps(report, indent=2))

Challenges & Solutions

  • subprocess with shell=True + user input is a command injection vulnerability
  • psutil requires installation (pip install psutil) — not in stdlib

Key Takeaways

  • Always use list form for subprocess.run() — never shell=True with user data
  • subprocess.check_output() is a convenient shorthand for capturing stdout
