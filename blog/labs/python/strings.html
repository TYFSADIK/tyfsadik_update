Step-by-Step Process

01
String Methods

Core string transformations.

s = '  Hello, World!  '
s.strip()           # 'Hello, World!'
s.lower()           # '  hello, world!  '
s.upper()           # '  HELLO, WORLD!  '
s.replace(',', ';') # '  Hello; World!  '
s.split(', ')       # ['  Hello', 'World!  ']

# Check
s.startswith('Hello')   # False (leading space)
s.strip().startswith('Hello')  # True
'hello'.isalpha()       # True
'123'.isdigit()         # True
02
Regex with re Module

Pattern matching and extraction.

import re

text = 'Call 416-555-1234 or 905-555-5678'
# Find all phone numbers
numbers = re.findall(r'\d{3}-\d{3}-\d{4}', text)
print(numbers)  # ['416-555-1234', '905-555-5678']

# Substitution
clean = re.sub(r'\s+', ' ', '  too   many   spaces  ')

# Named groups
m = re.match(r'(?P\w+) (?P\w+)', 'Alice Smith')
m.group('first')  # 'Alice'

Challenges & Solutions

  • Strings are immutable — s.replace() returns new string, doesn't modify s
  • re.match() only matches at start; re.search() searches anywhere

Key Takeaways

  • str.join() is much faster than concatenating with + in a loop
  • Compile regex patterns with re.compile() when using the same pattern repeatedly
