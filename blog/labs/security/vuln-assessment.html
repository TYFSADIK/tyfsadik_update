Vulnerability Assessment

Intermediate Security
NmapOpenVASNessusniktoCVSS

Objective

Perform vulnerability scanning with Nmap NSE scripts and nikto, interpret CVSS scores, and document findings.

Tools & Technologies

  • Nmap
  • OpenVAS
  • Nessus
  • nikto
  • CVSS

Key Commands

nmap -sV -sC -O target
nmap --script vuln target
nikto -h http://target
nmap -sV --script=vulners target

Architecture Overview

flowchart TD SCOPE[Define Scope\n& Authorization] --> DISCO[Discovery\nnmap -sn] DISCO --> PORTSCAN[Port Scan\nnmap -sV] PORTSCAN --> VULNSCAN[Vulnerability Scan\nnmap --script vuln] VULNSCAN --> WEBSCAN[Web Scan\nnikto / nikto2] WEBSCAN --> ANALYZE[Analyze Findings\nCVSS Scoring] ANALYZE --> REPORT[Report &\nRemediation Plan] style SCOPE fill:#1a1a2e,stroke:#ff4444,color:#ff4444 style REPORT fill:#1a1a2e,stroke:#00ff88,color:#e0e0e0

Step-by-Step Process

01
Discovery & Port Scanning

Identify live hosts and open services.

# Host discovery
nmap -sn 192.168.1.0/24          # ping sweep

# Service version detection
nmap -sV -sC -O 192.168.1.10     # versions + scripts + OS
nmap -sV -p- 192.168.1.10        # all 65535 ports
nmap -A 192.168.1.10             # aggressive scan
02
Vulnerability Scripts

Run NSE scripts to find known vulnerabilities.

# Run all vuln scripts
nmap --script vuln 192.168.1.10

# Check for specific CVEs
nmap -sV --script=vulners 192.168.1.10

# SMB vulnerabilities
nmap --script smb-vuln* -p 445 192.168.1.10
03
Web Application Scanning

Scan web servers for common vulnerabilities.

nikto -h http://192.168.1.10
nikto -h https://192.168.1.10 -ssl
nikto -h http://192.168.1.10 -output report.html

# Additional tools
whatweb http://192.168.1.10   # identify web tech

Challenges & Solutions

  • Vulnerability scanners require explicit written authorization — scanning without permission is illegal
  • False positives are common — always manually verify critical findings

Key Takeaways

  • CVSS score alone doesn't determine priority — consider exploitability and business impact
  • Document everything: scope, methodology, findings, evidence, and remediation recommendations
